# Thai Crop Price Sources

Official sources to search when farmers ask about current prices.

## Primary Sources

| Source | URL | What it covers |
|--------|-----|----------------|
| สำนักงานเศรษฐกิจการเกษตร (OAE) | https://www.oae.go.th | Daily/weekly wholesale prices for major crops: rice, cassava, sugarcane, rubber, maize, palm oil |
| กรมการค้าภายใน (DIT) | https://www.dit.go.th | Daily retail/wholesale commodity prices, regional price monitoring |
| ธนาคารเพื่อการเกษตรและสหกรณ์ (ธกส.) | https://www.baac.or.th | Reference purchase prices for paddy (ข้าวเปลือก) and rubber |
| ตลาดกลางยางพาราสงขลา | https://www.rubber.co.th | Rubber (ยางพารา) daily exchange prices |
| สมาคมโรงงานผู้ผลิตมันสำปะหลัง | — | Cassava (มันสำปะหลัง) starch factory gate prices |

## Recommended Search Queries

Construct queries like:
- `ราคา[ชื่อพืช] วันนี้ site:oae.go.th`
- `ราคา[ชื่อพืช] [จังหวัด] [ปี พ.ศ. ปัจจุบัน]`
- `[crop name] price Thailand today`

Example crops and their Thai names:
- ข้าวเปลือก — paddy rice
- มันสำปะหลัง — cassava
- อ้อย — sugarcane
- ยางพารา — rubber
- ข้าวโพด — maize / corn
- ปาล์มน้ำมัน — palm oil
- ทุเรียน — durian
- มะพร้าว — coconut

## Usage Notes

- OAE prices are wholesale/farm-gate — closest to what a farmer receives
- DIT prices skew toward retail markets — useful for comparison
- Prices vary by province; always note the region when reporting
- Always state the date of the price found — prices can be days old
