# Style Guide

> Edit this file to match your brand voice and content standards.
> This is injected into the AI context when generating any website content.

## Brand Voice

**Vaja AI** speaks like a knowledgeable colleague — warm, clear, and direct.

| Attribute | Description |
|-----------|-------------|
| **Tone** | Friendly and professional. Not corporate, not casual. |
| **Language** | Plain English. Avoid jargon. When Thai is appropriate, use it. |
| **Length** | Short sentences. Short paragraphs. Say more with less. |
| **Perspective** | User-first. Focus on outcomes, not features. |

## Do / Don't

| Do | Don't |
|----|-------|
| "Get AI that understands your work" | "Leverage cutting-edge artificial intelligence solutions" |
| "Share credits with your team" | "Enable collaborative credit pool management" |
| "Skills load automatically" | "Context-aware skill injection is performed" |
| "Try it free" | "Commence your complimentary trial period" |

## Headlines

- Use sentence case: "How AI helps Thai farmers" (not "How AI Helps Thai Farmers")
- Lead with the user benefit, not the product name
- Be specific: "Cut content time by half" beats "Save time"

## Thai Language Notes

- Use Thai for headlines targeting Thai-native audiences
- Product name: **วาจา AI** (Vaja AI)
- Keep English technical terms in English: AI, LINE OA, credits

## Content Lengths

| Content type | Recommended length |
|-------------|-------------------|
| Hero headline | 5–10 words |
| Hero subheadline | 1–2 sentences |
| Feature description | 1–3 sentences |
| Blog post | 400–800 words |
| Blog excerpt | 1–2 sentences, under 160 chars |
| Page section header | 3–7 words |

## Calls to Action

Preferred CTAs (in order of strength):
1. "Start for free"
2. "Try Vaja AI"
3. "Get started"
4. "Learn more"

Avoid: "Click here", "Submit", "Learn More About Our Solutions"
